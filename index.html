Senior Capstone Project -- The Adventuring Party Planner

An Android app for game masters.

View on GitHub

The Adventuring Party Planner allows a game master of any skill level to create an immersive fantasy world that any fan of tabletop roleplaying games would enjoy. From combat encounters with fearsome monsters to dynamic quests, the Adventuring Party Planner assists with the storybuilding aspect of running a successful tabletop campaign.

Encounter Daunting Monsters...

-Over 450 unique monsters
-Complete with combat stats
-Reward your players with great loot

Construct Inspiring Events...

-Over 200 trillion combinations
-Six captivating event categories

Bring NPCs to Life...

-Customize NPCs by race or class
-Detailed information, including occupation
